Необходимо реализовать web application, который позволяет :

1. ввести на web форме данные из json файла;
2. сохранить данные в хранилище по нажатию на кнопку на форме;
3. отобразить сохраненные в хранилище данные на форме так, чтобы можно было выполнить пункт 1 повторно.

Обязательные требования : embedded микроконтейнер (jetty, tomcat), postgres, maven, spring
